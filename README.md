# Amazon PPC Toolkit

> Open-source utilities for Amazon Sponsored Products campaign management.

**Coming Soon** - Under active development.

## Tools

### Bulk Sheet Parser
Parse and validate Amazon Advertising bulk upload sheets. Catch errors before uploading.

### Naming Convention Validator
Enforce consistent campaign naming across accounts. Configurable patterns, automatic detection of violations.

### Cross-Negation Generator
Automatically generate negative keyword and ASIN targeting to prevent campaign cannibalization.

### Search Term Analyzer
Categorize harvested search terms by intent, performance tier, and targeting recommendation.

## Who Is This For?

- Amazon PPC agencies managing multiple accounts
- Sellers running 50+ campaigns
- Anyone tired of manual bulk sheet errors

## Tech Stack

- Python 3.10+
- openpyxl (Excel processing)
- CLI + importable modules
